📦

Stock Tracking

Track stock quantities and inventory movements in real time.

Installation

Quick and simple installation process.

✔ Download the module and extract it into your Odoo addons directory.
✔ Restart your Odoo server.
✔ Enable Developer Mode in Odoo.
✔ Navigate to Apps → Update Apps List.
✔ Search for Inventory Indent Management.
✔ Click Install to activate the module.

Configuration

Configure users, permissions, and approval workflows.

✔ Configure Inventory Indent User access rights.
✔ Configure Inventory Indent Manager permissions.
✔ Assign users based on department workflows.
✔ Define approval hierarchy using security groups.
✔ Manage inventory request approvals and validations efficiently.
