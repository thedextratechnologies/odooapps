Installation

Quick setup process for Inventory Indent Management.

✔️ Download the module and extract it into your Odoo addons directory.
✔️ Restart your Odoo server.
✔️ Enable Developer Mode in Odoo.
✔️ Navigate to Apps → Update Apps List.
✔️ Search for Inventory Indent Management.
✔️ Click Install to activate the module.

Configuration

Configure users, permissions, and approval workflows easily.

✔️ Configure Inventory Indent User access rights.
✔️ Configure Inventory Indent Manager permissions.
✔️ Assign users based on department workflows.
✔️ Define approval hierarchy using security groups.
✔️ Manage inventory request approvals and validations efficiently.
